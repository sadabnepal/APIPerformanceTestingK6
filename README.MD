### Getting Started
```
1. git clone https://github.com/sadabnepal/APIPerformanceTestingK6.git
2. Navigate to APIPerformanceTestingK6
```

```

npm run testPostCall
npm run testPutCall
npm run testPatchCall
npm run testWithVirtualUsers
npm run test
```
